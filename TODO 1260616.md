# TPT Cortex ‚Äî Task Checklist

## Phase 1: Core Compiler (Weeks 1‚Äì3)
> **Milestone:** `cortex compile hello.ctx` outputs valid JSON AST; invalid code rejected with line/column errors.

### Monorepo Scaffold
- [ ] Create root `Cargo.toml` as Rust workspace (members: `cortex-engine`, `cortex-shell`)
- [ ] Create `cortex-engine/Cargo.toml` with deps: `clap`, `serde`, `serde_json`, `thiserror`
- [ ] Create `go.work` + `cortex-daemon/go.mod` (stub only)
- [ ] Add `.gitignore` (Rust, Go, Node, Android targets)
- [ ] Add `README.md`

### Formal Grammar
- [ ] Write `cortex-engine/GRAMMAR.ebnf` ‚Äî canonical EBNF covering all tokens and productions

### Lexer (`cortex-engine/src/lexer/`)
- [ ] Define `Token` enum (keywords, types, literals, operators, delimiters, `NATIVE_DOT`)
- [ ] Define `Span { line, col, len }` struct
- [ ] Implement `Lexer::new(source: &str)` and `Lexer::tokenize() -> Vec<Spanned<Token>>`
- [ ] Handle `native.` as a special `NATIVE_DOT` token
- [ ] Return precise `LexError` with span on invalid input
- [ ] Unit tests: keywords, string escapes, integer/float literals, `native.` prefix, error cases

### AST (`cortex-engine/src/ast/`)
- [ ] Define all AST node types with `Span` fields
  - [ ] `Program`, `Task`
  - [ ] `LetStmt`, `IfStmt`, `ReturnStmt`, `ExprStmt`
  - [ ] `BinaryExpr`, `UnaryExpr`, `CallExpr`, `NativeCallExpr`, `IndexExpr`
  - [ ] `Literal`, `Ident`, `TypeAnnotation`
- [ ] Derive `serde::Serialize` on all nodes for JSON output

### Parser (`cortex-engine/src/parser/`)
- [ ] Implement recursive descent for declarations and statements
- [ ] Implement Pratt (precedence climbing) for expressions
- [ ] Collect multiple `ParseError { span, message }` before aborting
- [ ] Unit tests: valid programs, operator precedence, error recovery

### Semantic Analyzer / Type Checker (`cortex-engine/src/checker/`)
- [ ] Implement `SymbolTable` with lexical scoping (push/pop scope)
- [ ] Type-check `let` declarations and assignments
- [ ] Type-check function call arguments and return types
- [ ] Validate `native.*` calls against `PermissionManifest`
- [ ] Return `TypeError` and `PermissionError` variants with spans
- [ ] Unit tests: type mismatches, undeclared variables, permission violations

### CLI (`cortex-engine/src/main.rs`)
- [ ] `cortex compile <file>` ‚Äî lex ‚Üí parse ‚Üí type-check ‚Üí print JSON AST to stdout
- [ ] `cortex compile <file> --emit=ast` ‚Äî explicit flag (same as default)
- [ ] `cortex compile <file> --allow=native.log,native.fs.read` ‚Äî permission manifest flag
- [ ] Error output format: `error[E001]: type mismatch ‚Äî expected i32, found string (line 4, col 12)`
- [ ] Exit code 0 on success, 1 on any error

### Phase 1 Verification
- [ ] `cortex compile hello.ctx --allow=native.log` ‚Üí valid JSON AST
- [ ] `cortex compile bad_types.ctx` ‚Üí type error with line number
- [ ] `cortex compile sneaky.ctx` (calls `native.fs.write` without `--allow`) ‚Üí permission error

## Phase 2: Virtual Machine (Weeks 4‚Äì6)
> **Milestone:** `cortex run math.ctx` executes pure math/logic and prints result.

### Bytecode Compiler (`cortex-engine/src/compiler/`)
- [ ] Define `Instruction` enum: `PUSH_I32`, `PUSH_F64`, `PUSH_STR`, `PUSH_BOOL`, `LOAD`, `STORE`, `ADD`, `SUB`, `MUL`, `DIV`, `EQ`, `NEQ`, `LT`, `LTE`, `GT`, `GTE`, `AND`, `OR`, `NOT`, `CALL_NATIVE`, `JUMP`, `JUMP_IF_FALSE`, `RETURN`, `HALT`
- [ ] Define `Chunk { instructions: Vec<Instruction>, string_table: Vec<String> }`
- [ ] Implement `Compiler::compile(ast: Program) -> Result<Chunk>`
- [ ] Text bytecode emitter (`--emit=asm` ‚Üí human-readable `.ctxasm`)
- [ ] Binary bytecode writer (`-o out.ctxb`)

### VM (`cortex-engine/src/vm/`)
- [ ] Implement value stack (`Vec<Value>`)
- [ ] Implement call stack for task frames
- [ ] Op budget counter: default 10,000 ops, configurable via `--ops-limit`
- [ ] `ExecutionError::Timeout` on budget exhaustion
- [ ] `NativeRegistry` trait object for dispatching `CALL_NATIVE` instructions
- [ ] Built-in `native.log` implementation for CLI testing

### CLI additions
- [ ] `cortex run <file>` ‚Äî compile + execute in one step
- [ ] `cortex compile <file> --emit=asm` ‚Äî print text bytecode
- [ ] `cortex compile <file> -o out.ctxb` ‚Äî write binary file

### Phase 2 Verification
- [ ] `cortex run math.ctx` (e.g. `let x: i32 = 6 * 7; native.log(x)`) ‚Üí prints `42`
- [ ] `cortex compile math.ctx --emit=asm` ‚Üí human-readable bytecode listing
- [ ] Infinite loop script ‚Üí `ExecutionError::Timeout` after op budget
